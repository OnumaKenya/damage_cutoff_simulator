# Gibbs 振動とフィルタ: Fejér 平均・Lanczos σ 因子の要否

## 1. 問題設定

`docs/gilpelaez.md` では「不連続(ジャンプ)を持つ関数をそのままフーリエ反転すると、ジャンプの周りで **Gibbs 振動**が立って壊れる」という現象を見た。中心化 CDF $g(y) = \mathbb{1}_{y>0} - F(y)$ が平均で大きさ 1 の段差を持ち、$(\varphi-1)/(it)$ が $1/t$ でしか減衰せず、有限打ち切りで平均のど真ん中が振動する、という現象である。`docs/discrete.md` では離散(格子)分布の「密度」が Dirac デルタの和になり、密度版 COS が全原子で Gibbs を起こす、という別の不連続も見た。

不連続データの Gibbs を抑える古典的な道具に、**Fejér 平均(Cesàro 総和)**と **Lanczos の σ 因子(σ 近似)**がある。本稿は「ブルアカのこの問題系でこれらを使うべきか」を考察する。

結論を先に述べる。

- Fejér 平均も Lanczos σ も、本質は**フーリエ係数(= 特性関数の評価値)に窓 $\sigma(u)$ を掛ける**操作である。$\sigma$ が実正値ならこれは**分布をある平滑化カーネルで畳み込む**ことに等しく、特に **Lanczos σ = 小さな一様分布の畳み込み**、**Fejér = Fejér 核(三角形分布系)の畳み込み**という、本プロジェクトの「一様分布の混合」描像にそのまま乗る綺麗な解釈を持つ。
- Fejér はカーネルが非負なので**オーバーシュート完全消失(CDF の単調性を保つ)**だが、ぼけ幅 $\sim 2\pi/U$・収束が $O(1/N)$ と遅い。Lanczos σ はオーバーシュートを $\approx 9\% \to \approx 1\%$ に減らしつつ分解能はそこそこ保つが、やはり代数的精度どまり。
- ところが**この問題系では原理的にどちらも不要**である。ジャンプの**位置と大きさが解析的に既知**(CDF なら平均での段差 1、格子なら格子点)なので、`docs/gilpelaez.md` のように**段差を閉形式で引く(Gil-Pelaez の $1/2$)**ほうが圧倒的に優れる——フィルタは「振動」を「全域の遅い収束」と引き換えにするのに対し、解析的除去はスペクトル精度(機械精度)を保つ。
- フィルタが本当に効くのは「**ジャンプの位置・大きさが未知で解析的に引けない**」「**とりあえず単調で見栄えのよい曲線が欲しい**(可視化)」場合に限られる。そのときでも Fejér / Lanczos(フィルタ次数 1〜2)より、**高次の指数型フィルタ**のほうが不連続から離れた領域でスペクトル精度を回復するので筋がよい。

記法は `docs/edge.md`・`docs/saddlepoint.md`・`docs/gilpelaez.md` を引き継ぐ。$F$ を $S_n$ の分布関数、$\varphi_{S_n}$ を特性関数、$u_k = k\pi/(b-a)$ を COS の周波数格子、$U := u_{N} = N\pi/(b-a)$ を打ち切り周波数とする。

## 2. そもそも不連続はどこにあるか

`docs/gilpelaez.md`・`docs/discrete.md` の整理を借りると、この問題系で Gibbs を起こしうる不連続は次の三種である。

1. **CDF のジャンプ**(`docs/gilpelaez.md`)。中心化 CDF $g$ が平均で大きさ 1 の段差。
2. **格子分布のデルタ**(`docs/discrete.md`)。離散一様の和の「密度」が格子点に集中したデルタの和。
3. **1 Hit の箱型密度**。$n=1$ の一様分布は密度が両端で不連続(箱)。和を取ると $n$ 個で密度は区分多項式 $C^{n-2}$ になり、$n$ が増えるほど滑らかになる(`docs/gilpelaez.md` の $F_k \sim k^{-n}$)。

逆に言えば、**Hit 数が多い bulk の密度・CDF はすでに滑らか**で、そこに Gibbs はない。フィルタが意味を持つのは上の不連続が前面に出る場面だけ、という点をまず押さえる。

## 3. Fejér 平均(Cesàro 総和)

フーリエ部分和 $S_N(x)$ の Fejér 平均は、部分和の **Cesàro 平均**

$$
\sigma_N(x) = \frac{1}{N}\sum_{m=0}^{N-1} S_m(x)
$$

である。係数の立場で見ると、第 $k$ 係数に**三角窓** $\big(1 - k/N\big)$、すなわち周波数で

$$
W_{\text{Fej}}(u) = \Big(1 - \frac{|u|}{U}\Big)\,\mathbb{1}_{|u|<U}
$$

を掛ける操作である。この窓の逆変換は **Fejér 核**

$$
\mathcal{K}_{\text{Fej}}(x) = \frac{1}{2\pi}\int_{-U}^{U}\Big(1-\tfrac{|u|}{U}\Big)e^{-iux}\,du = \frac{U}{2\pi}\,\operatorname{sinc}^2\!\Big(\frac{Ux}{2}\Big) \ \ge 0
$$

で、**非負**である。非負カーネルでの畳み込みは最大値原理を満たすので、$\sigma_N$ は**オーバーシュートを一切持たず**、連続区間で一様収束する。CDF に使えば**単調性が保たれる**という利点がある。

代償は二つ。第一に、ジャンプを幅 $\sim 2\pi/U = 2(b-a)/N$ にわたって**なだらかに塗りつぶす**(段差が傾斜になる)。第二に、収束が連続点でも $O(1/N)$ と遅い($\operatorname{sinc}^2$ 核の $x^{-2}$ 裾が効く)。要するに**強い低域通過**で、安全だが鈍い。

## 4. Lanczos の σ 因子(σ 近似)

Lanczos の σ 因子は、第 $k$ 係数に

$$
\sigma_k = \operatorname{sinc}\!\Big(\frac{k}{N}\Big) = \frac{\sin(\pi k/N)}{\pi k/N}, \qquad \text{すなわち}\quad \sigma(u) = \operatorname{sinc}\!\Big(\frac{u}{U}\Big)
$$

を掛ける操作である。これは部分和 $S_N$ を**最高次モードの 1 波長ぶん($\Delta = (b-a)/N$)で局所平均**することに等しい。

$$
\widetilde S_N(x) = \frac{1}{2\Delta}\int_{x-\Delta}^{x+\Delta} S_N(t)\,dt.
$$

オーバーシュートは Gibbs 定数の $\approx 8.95\%$ から $\approx 1\%$ 程度まで減り、しかも Fejér ほど分解能を潰さない(段差はそこそこ立ち上がる)。「振動はほぼ消したいが、なまし過ぎたくない」中間策である。

ただし Lanczos は**ハードな打ち切り $\mathbb{1}_{|u|<U}$ を残したまま** sinc テーパを掛けるので、実効窓 $\operatorname{sinc}(u/U)\,\mathbb{1}_{|u|<U}$ のカーネルは Dirichlet 核との畳み込みになり**非負ではない**——だから小さなオーバーシュートが残る。Fejér が完全消失なのは、三角窓が $u=U$ で滑らかに 0 に達してカーネルが非負になるから、という対比である。

## 5. 統一視点: 係数への窓 = CF への乗算 = 分布の畳み込み

ここがこの問題系で一番効く見方である。COS 法で「第 $k$ 係数に窓 $\sigma(u_k)$ を掛ける」ことは、**特性関数を $\varphi_{S_n}(u)\,\sigma(u)$ に置き換える**ことに他ならない。そして $\sigma$ が**ある確率分布の特性関数(実正値で $\sigma(0)=1$)**であれば、CF の積は畳み込みなので、これは

$$
S_n \ \longrightarrow\ S_n + \xi, \qquad \xi \sim (\text{カーネル } \sigma \text{ に対応する独立な微小ノイズ})
$$

という**分布の平滑化(独立な微小ノイズの付加)**そのものである。具体的に対応を書くと:

- **Lanczos σ** $\sigma(u)=\operatorname{sinc}(u/U)=\dfrac{\sin(\pi u/U)}{\pi u/U}$ は、幅 $2\pi/U = 2\Delta$ の**一様分布** $U(-\pi/U,\,\pi/U)$ の特性関数。つまり Lanczos σ 平滑化 $=$ **ダメージに幅 2 セルぶんの一様ノイズを 1 個足す**こと。
- **Fejér** $W_{\text{Fej}}(u)=(1-|u|/U)$ は **Fejér 核**($\operatorname{sinc}^2$、三角形分布系)の特性関数。つまり Fejér 平滑化 $=$ **三角形分布(= 一様 2 個の和)で畳み込む**こと。

本プロジェクトはそもそもダメージを**一様分布の混合の和**として扱っているのだから、「もう一個小さい一様を足す」という操作は描像の中に自然に収まる。しかもこれは `docs/discrete.md` の議論と地続きである——格子幅 $g$(= もう一つの微小スケール)が分布の広がり $s_n$ に比べて十分小さければ無視できた、というあの話と同じで、**平滑化カーネルの幅 $2\pi/U \ll s_n$ である限り、足したノイズは bulk を実質動かさない**(Sheppard 的な $O((2\pi/U)^2)$ の分散バイアスだけ残る)。

逆に言えば、フィルタは「無料の平滑化」ではなく**分布そのものを微小に変える(バイアスを入れる)**操作だ、という認識が大事である。$\sigma$ が実正値カーネルの CF である限り、得られる結果は**ちゃんとした(平滑化された)確率分布**(CDF は単調・$[0,1]$ 値)になる——この健全性が Fejér/Lanczos の取り柄である。

## 6. この問題系では原則「不要」

では使うべきか。**原則として不要**、というのが結論である。理由は `docs/gilpelaez.md` の主題そのものである。

フィルタは「ジャンプの位置も大きさも知らない」前提で振動を**力ずくで**なます汎用手段である。ところがこの問題系では、

- CDF のジャンプは**平均(中心化原点)で大きさ 1** と**解析的に既知**、
- 格子のデルタは**格子点上**と**既知**、

なので、ジャンプを**閉形式で引いてしまう**ほうが断然よい。それが Gil-Pelaez の $\dfrac12$ ステップ(= COS の線形ランプ)であり、引いた残りは $\varphi/t \sim t^{-(n+1)}$ で**スペクトル的に速く**減衰した(`docs/gilpelaez.md`: 132 Hit で数百項・機械精度)。

両者の精度は桁が違う。

| 方針 | 不連続近傍 | 滑らかな領域 | CDF 単調性 |
|---|---|---|---|
| 素朴打ち切り(矩形窓) | Gibbs $\approx 9\%$ | 速いが振動が混入 | 崩れる |
| Fejér | 振動なし・幅 $2\pi/U$ で塗る | $O(1/N)$ | 保たれる |
| Lanczos σ | 振動 $\approx 1\%$ | 代数的どまり | ほぼ保たれる |
| **解析的除去(Gil-Pelaez / COS)** | **そもそもジャンプを持ち込まない** | **$t^{-(n+1)}$、機械精度** | **構造的に単調** |

フィルタは「振動」というローカルな病を「全域の遅い収束」というグローバルな代償で買う。**位置のわかっているジャンプを 1 本引くだけで済むのに、全係数を鈍らせる必要はない**、という整理である。`docs/discrete.md` の格子の場合も同様で、**1 周期上の逆 DFT で PMF が厳密に出る**(打ち切り誤差ゼロ)ので、密度版 COS を Fejér で延命する動機はない。

## 7. それでもフィルタが有用な場面

汎用手段である以上、出番がないわけではない。

1. **ジャンプを解析的に引けないとき。** 例えば、`docs/edge.md` の前提を外れる「ダメージストッパーによる遮断(cutoff)」や HP 依存倍率で、和の分布に**位置や大きさが閉形式で書けない不連続・原子**が出る場合。cutoff そのものは閾値が既知なので原子位置 $=$ 閾値・質量 $= P(\text{素のダメージ}\ge \text{閾値})$ と解析的に引けるが、状態変化が絡んで非自明になる場合は、Lanczos σ で**とりあえず数値を安定させる**価値がある。
2. **可視化・速報。** 解析的なステップ処理を実装する手間をかけず、**単調で見栄えのよい CDF / なめらかな PMF ヒストグラム**をさっと描きたいとき。Lanczos σ は係数に sinc を一行掛けるだけで、オーバーシュートが消えてグラフが落ち着く。精度を要求しない図には十分。
3. **離散 PMF の平滑表示。** `docs/discrete.md` の厳密 PMF はギザギザの棒である。連続的な「なめらかな密度」として見せたいなら、Fejér 核(非負)で畳み込めば負値の出ない平滑密度になる(中身は「一様 2 個ぶんなまし」)。あくまで表示用で、計算の本筋は厳密 PMF。
4. **診断。** フィルタ前後で結果が大きく動くなら、それは「打ち切り $U$ の手前に無視できない高周波(= 鋭い構造 / 原子 / 格子)が残っている」サイン。`docs/discrete.md` のエイリアス位置 $2\pi/g$ に $U$ が近づいていないかの点検に使える。

そして**もしフィルタを使うなら**、Fejér(フィルタ次数 1)や Lanczos σ(次数 〜2)より**高次フィルタ**を推す。指数型フィルタ $\sigma(u)=\exp\!\big(-\alpha (u/U)^{2p}\big)$ や Vandeven フィルタは、端点で $p-1$ 階まで 0 になるよう作られ、**不連続から離れた領域でスペクトル精度を回復**しつつ Gibbs を殺す(不連続からの距離 $d$ に対し誤差 $\sim d^{-(p-1)}$)。Fejér/Lanczos は「素朴・古典」だが、回復力では高次フィルタに劣る。とはいえ繰り返すと、**位置既知のジャンプはフィルタより解析的除去が常に優る**。

## 8. まとめ

- Fejér 平均・Lanczos σ は、フーリエ係数 $=$ 特性関数評価値に窓 $\sigma(u)$ を掛ける操作で、**$\varphi_{S_n}\to\varphi_{S_n}\sigma$、すなわち分布を平滑化カーネルで畳み込む**ことと同義。**Lanczos σ = 小さな一様 1 個の畳み込み**、**Fejér = 三角形分布(一様 2 個)の畳み込み**——本プロジェクトの一様混合描像にそのまま乗る。
- Fejér は非負カーネルで**オーバーシュート完全消失・CDF 単調**だが $O(1/N)$ と鈍い。Lanczos σ はオーバーシュートを $9\%\to1\%$ に減らすが代数的精度どまり。いずれも「振動」を「全域の遅い収束」と引き換えにする。
- **この問題系では原則不要。** ジャンプの位置・大きさが解析的に既知なので、`docs/gilpelaez.md` の段差除去(Gil-Pelaez の $1/2$、COS の線形ランプ)や `docs/discrete.md` の 1 周期逆 DFT のほうが、機械精度かつ単調性込みで圧倒的に優る。
- フィルタが効くのは「ジャンプを解析的に引けない」「可視化・速報」「離散 PMF の平滑表示」「診断」に限る。使うなら Fejér/Lanczos より**高次の指数型 / Vandeven フィルタ**。それでも位置既知のジャンプは解析的除去が常に勝つ。
